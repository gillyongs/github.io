익성비는 메이플스토리에서 레벨을 1부터 10 사이 랜덤한 값만큼 올려주는 아이템입니다. <br><br>
현재 레벨에서 익성비를 n개 사용하였을때 자신의 운이 상위 몇%인지 계산해주는 백분율 시뮬레이터와 <br><br>
현재 레벨에서 n 레벨을 달성하기 위해 몇개의 익성비를 사용해야하는지 계산해주는 기대값 시뮬레이터입니다.<br>
